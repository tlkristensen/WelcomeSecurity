# Hvad er penetrationstesting?

Penetrationstesting er en kontrolleret simulering af et hackerangreb mod din virksomheds IT-systemer for at identificere sårbarheder, før de rigtige kriminelle opdager dem.

## Vores penetrationstest proces

### 1. Planlægning og rekognoscering
- Definering af test scope og mål
- Indsamling af information om målsystemerne
- Identificering af potentielle indgangspunkter

### 2. Scanning og enumeration
- **Port scanning** for at identificere åbne tjenester
- **Vulnerability scanning** med professionelle værktøjer
- **Service enumeration** for detaljeret systemanalyse

### 3. Udnyttelse af sårbarheder
- Kontrolleret exploitation af identificerede sårbarheder
- Eskalering af privilegier hvor muligt
- Bevægelse lateralt i netværket

### 4. Post-exploitation
- Vurdering af adgang til kritiske systemer
- Test af data exfiltration muligheder
- Dokumentation af påvirkning

> Penetrationstesting giver dig indsigt i hvordan en rigtig angriber ville tænke og handle

## Test typer

### Eksterne penetrationstest
- Test fra internettet mod din virksomheds offentlige systemer
- Simulerer angreb fra eksterne trusler
- Fokus på perimeter forsvar

### Interne penetrationstest  
- Test fra virksomhedens interne netværk
- Simulerer insider trusler eller kompromitterede systemer
- Evaluerer lateral movement muligheder

### Wireless penetrationstest
- Test af WiFi sikkerhed og konfiguration
- Rogue access point detection
- WPA/WPA2/WPA3 sikkerhedsevaluering

### Social engineering test
- Test af medarbejdernes sikkerhedsbevidsthed
- Phishing simulationer
- Physical security evaluering

## Leverancer

### Detaljeret rapport
Omfattende teknisk rapport med:
- Executive summary til ledelsen
- Tekniske detaljer for IT-afdelingen
- Risikovurdering og prioritering
- Proof-of-concept for kritiske sårbarheder

### Remediation guidance
- Specifikke anbefalinger til at lukke sårbarheder
- Implementeringsguides og best practices
- Prioriteret handlingsplan

### Re-test service
- Verifikation af at sårbarheder er blevet lukket
- Reduceret pris for opfølgende test
- Løbende sikkerhedsevaluering
